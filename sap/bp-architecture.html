址 BUT020 — 地址数据

ADDRNUMBER ADDR_TYPE ADDR_ORIGIN VALID_FROM VALID_TO XDFADR
引用 ADRC（地址）、ADR6（邮件）、ADR2（电话）等标准地址表

联 BUT021 — 联系人关系

RELATION_ID RELTYP RELNR VALID_FROM VALID_TO FUNCTION
BP 与其他 BP 的关系（如联系人→公司、员工→部门）

讯 ADRC / ADR6 / ADR2

ADRC: 地址 ADR6: 邮箱 ADR2: 电话 ADR3: 传真 ADR12: FTP/URL
标准 SAP 通讯数据表，通过地址编号（ADDRNUMBER）与 BUT020 关联
2. BP Role 层（角色分类）— 决定 BP 在哪些业务场景中使用
FI
财务供应商
FLVN00 · Supplier (FI)
数据归属：Company Code 层
  • 对账科目（AKONT）
  • 付款条件（ZTERM）
  • 催款程序（MAHNA）
  • 付款方式（ZWELS）
  • 预扣税（QSREC）
← 兼容 ECC →
LFB1（供应商公司代码）
PU
采购供应商
FLVN01 · Supplier Purchasing
数据归属：Purchasing Org 层
  • 采购币种（WAERS）
  • 国际贸易条款（INCO1）
  • 自动 PO（BSART）
  • 最小订单金额（MINBW）
  • 计划交货时间（PLIFZ）
← 兼容 ECC →
LFM1（供应商采购组织数据）
CF
财务客户
FLCU00 · Customer (FI)
数据归属：Company Code 层
  • 对账科目（AKONT）
  • 付款条件（ZTERM）
  • 催款程序（MAHNA）
  • 容差组（TOGRU）
  • 预扣税（WT_WITHCD）
← 兼容 ECC →
KNB1（客户公司代码数据）
SD
销售客户
FLCU01 · Customer Sales
数据归属：Sales Org 层
  • 销售区域（VKORG/VTWEG）
  • 定价程序（KALKS）
  • 装运条件（VSBED）
  • 交货工厂（DWERK）
  • 客户统计组（VERSG）
← 兼容 ECC →
KNVV（客户销售区域数据）
3. 银行信息 — 兼容 Vendor Bank
BP 银行详情 → 兼容 ECC 供应商银行主数据
LFBK Bank Key (BANKK) Account No. (BANKN) IBAN SWIFT/BIC Partner Bank Type Bank Control Key Reference Details
关联路径：
BP → BUT0BK → BUT0BANK → LFBK（兼容视图）
支持多银行账户，区分自有银行 / 伙伴银行类型
